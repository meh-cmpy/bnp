# BNP Eventservice

Homepage für BNP Eventservice — Dachmarke von Cocktailfreunde (mobile Cocktailbar) und Fruchtfreunde (Smoothie Catering).

## Struktur

- `index.html` — Startseite
- `impressum.html` — Impressum
- `datenschutz.html` — Datenschutzerklärung
- `cocktailbar-gamescom-koeln.html` — Beispiel-Landingpage für Messeauftritte (gamescom Köln)
- `assets/` — Logos und Favicon
- `robots.txt`, `sitemap.xml` — SEO-Grundlagen
- `CNAME` — Custom-Domain-Konfiguration für GitHub Pages
